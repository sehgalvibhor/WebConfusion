Text Editor

Basically a text editor is a program used to edit text files like documents,source code of programs etc.
There exist text editors with every OS and software development packages.
    Some examples of basic text editors are:
  • Write(Sugar)
  • nano(Terminal Activity)
  • vi(Terminal Activity)
  • Gedit(Sugar)

    • Although we do have a plethora of text editors available but we have to choose just one.In this tutorial we would be using Write Activity.
    You have a freedom of selecting any text editor of your choice.

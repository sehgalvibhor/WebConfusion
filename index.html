INTERMEDIATE CHALLENGES

SVG Challenges
These challenges show you how to make pictures using SVG web technogology. SVG stands for Scalar Vector Graphics. There are really two ways to make a picture with a computer. One is to supply a row by row list of colors for each pixel. The other is to tell the computer how to make the picture. SVG is an example of this second way.
